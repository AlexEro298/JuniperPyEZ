# To use this project, need:

1. Use command:
* git clone https://github.com/AlexEro298/JuniperPyEZ
* cd JuniperPyEZ
* python -m venv ./venv
* . venv/bin/activate;
* pip install -r requirements.txt
* python save_config.py
* deactivate;

2. Create a file with the name: 
* authentification.py

3. In the file write:
* username = ''
* password = ''
* ip_device = ['', '']
